# 服务器框架指南

author@Aloudname

你想知道技术部平时是如何~~运营维护~~服务器的吗？亦或者，你想学习如何方便快速地搭建、管理自己的服务器吗？

本文件夹 `docs/server_arch`提供了一些关于服务器运行、维护（即**服务器运维**）所需的一系列工具/技术的介绍和教程，包括用于**公网转发**的frp：[公网转发配置不完全指南](./docs/server_arch/env_cfg/frp/FRP-doc.md)、**图形化管理面板**MCSM：[MCSM开服简要流程](./docs/server_arch/env_cfg/mcsm/mcsm.md)，以及**PVE网络转发**：[pve网络转发](./docs/server_arch/env_cfg/pve/网络转发.md)。除此之外，还包括。

本文档作为server_arch文件夹中诸多眼花缭乱技术/工具的总览，带初入此门的你一起了解一下这些技术/工具的作用，以及为什么要用到这种技术/工具。我相信，解决完“**为什么这么做**”的疑惑后再去细致了解每项技术，你能对它们有更深刻的理解和更长久的记忆。
